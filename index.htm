Muchat - A web version of WhatsApp using Socket.io and NodeJS
HTML, CSS, JAVASCRIPT, Socket.io, NodeJS

Muchat is a web version of WhatsApp except you only need a username and password to register and use the application. The application has the same features as WhatsApp and supports private chat conversations. Users are able to also send video, audio, images and documents do their Muchat contacts. This application developed using socket.io, nodejs and express js.

To register on the app, visit http://muchat-env.eba-gzktp4f7.us-east-2.elasticbeanstalk.com/ and create an account with a username and password

Muchat is not backed by any database but your informations are safely stored and protected
The application is currently hosted on amazon elasticbeanstalk service.

Chat-App with Socket.io and NodeJS
HTML, CSS, JAVASCRIPT, Socket.io, NodeJS

This is a simple chat application developed using socket.io and nodejs that enable users that are currently online to exchange messages. The application is in two folds. General and Private chat.

To have a private chat with a user, visit https://my-socket-io-chat-app.herokuapp.com/login
